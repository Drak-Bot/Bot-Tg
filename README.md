# Project Documentation

## Project Description
This project is a Telegram bot designed to enhance user interaction and provide automated responses based on various commands. It's built to be user-friendly and flexible for a variety of use cases.

## Features List
- User authentication
- Command handling
- Message responses based on user input

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Drak-Bot/Bot-Tg.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Bot-Tg
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage Guide
To start the bot, run the following command:
```bash
npm start
```

## Available Commands
- `/start`: Begins the interaction with the bot.
- `/help`: Provides a list of available commands and their descriptions.

## Setup Configuration Steps
1. Create a `.env` file in the project root and set your Telegram bot token:
   ```plaintext
   TELEGRAM_BOT_TOKEN=your_token_here
   ```
2. Configure any other necessary environment variables as per the requirements.
